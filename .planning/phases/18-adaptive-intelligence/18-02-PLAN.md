# Phase 18 Gap Plan: Empty Suffix Tokens Fix

<objective>
Fix an issue where `LLMEngine.generateCompletion` crashes with an `Index out of range` fatal error in `MLXArray+Indexing.swift` because it attempts to pass an empty `suffixTokens` array to `MLXLMCommon.generate`.
</objective>

<verification>
## Verification Criteria
- `LLMEngine.swift` contains a guard statement checking if `suffixTokens.isEmpty` before passing it to `MLXLMCommon.generate`.
- The generate stream should return early with an empty string when there are 0 suffix tokens instead of crashing.
</verification>

<must_haves>
- Add `guard !suffixTokens.isEmpty else { return "" }` immediately after `let suffixTokens = Array(fullTokens[safePrefix...])`.
</must_haves>
