# Features Research

## Table Stakes
- System-wide text field monitoring
- Local on-device inference (<150ms)
- Accept suggestion with Tab
- Menu bar icon with quick disable toggles
- Launch at login

## Differentiators
- Screen OCR context via Vision framework
- Clipboard context awareness
- Custom user instructions and tone profiles
- Snippet auto-detection and expansion
- Per-app custom settings (model, length, tone)

## Anti-Features
- Cloud syncing of history
- Third-party API keys for inference
- Subscription requirements
